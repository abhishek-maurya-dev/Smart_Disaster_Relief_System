============================================================
                PROJECT OVERVIEW
============================================================

A complete disaster relief management system built with Python.

Features:
- Disaster Management (CRUD)
- Victim Management
- Volunteer Management
- Relief Camp Management
- Inventory Management (Food & Medicine)
- Report Generation
- Data Backup & Restore
- Admin Authentication

============================================================
                TECHNOLOGY STACK
============================================================

Language: Python 3.9+
Storage: CSV Files
Logging: Python Logging Module
Authentication: SHA256 Hashing
